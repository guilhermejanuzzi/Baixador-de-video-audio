# Baixador de Vídeo/Áudio

Aplicação em Python com interface gráfica usando CustomTkinter.

## Funcionalidades
- Download de vídeos (melhor qualidade)
- Download de áudio
- Histórico de downloads
- Integração com ffmpeg

## Requisitos
- Python
- yt-dlp
- customtkinter
- ffmpeg
